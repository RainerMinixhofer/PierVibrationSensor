# Data Directory

This `data` directory contains datasets and files relevant to the Pier Vibration Sensor project. The contents are used for analysis, testing, and validation of the vibration sensor system.

## Contents

- **Raw Sensor Data:** Collected vibration measurements from the sensor hardware.
- **Processed Data:** Cleaned and formatted data sets for further analysis.
- **Sample Files:** Example data files for testing and demonstration purposes.
- **Documentation:** Any notes or metadata describing the data formats and collection methods.
- **Security data** Credential for logging into WLAN
- **Frontend data** HTML frontend file "chart.html" which is served through the webserver code of the Pier Vibration Sensor project.
